# FifthModuleProject
Проект пятого модуля

Реализованы следующие методы:
	1. CheckStr
		Данный метод не получает никаких параметров на вход
		Работа: Пользователь вводит строку
				Метод обрезает строку по бокам от пробелов
				Проверяет строку на пустоту, на то, что строка может быть числом
				Требует выполнения условий до победного
		Возвращает строку

	2. CheckInt
		Данный метод получает на вход:
			Строку-сообщение
			Булев, указывающий на то, необходимо ли нам именно положительное число. Или возможен 0
		Работа: Проверяет строку на возможность конвертировать в Int
				Проверяет на то, число положительное, число может быть нулем?
				Требует выполнения условий до победного
		Возвращает число

	3. CheckBool
		Данный метод получает на вход строку-сообщение
		Работа: Проверяет строку на то, подходит ли она под булев тип данных (6 вариантов: да, Да, ДА, нет, Нет, НЕТ)
				Требует выполнения условий до победного
		Возвращает булев тип данных

	4. EnterArray
		Данный метод получает на вход число элементов массива
		Работа: Заполняет массив, используя проверки строк методом CheckStr
		Возвращает массив

	5. EnterUser
		Данный метод ничего не получает на вход
		Работа: создает и заполняет кортеж данных пользователя, используя методы 1-4
		Возвращает кортеж

	6. PrintTuple
		Данный метод получает на вход кортеж данных пользователя
		Работа: Печатает каждый элемент кортежа. 
				Если нет питомцев или любимых цветов (массивы пустые), то уведомляет об этом
		Ничего не возвращает
